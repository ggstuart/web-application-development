The CSS box model describes every element as a rectangular box consisting of four layers: the content area (width and height set by content or explicit dimensions), surrounded by padding (space between content and border), then the border (which can have thickness, style, and color), and finally margin (outer space separating the element from neighbors). The default box-sizing: content-box means width/height properties apply only to the content area (padding + border add to total size), while box-sizing: border-box includes padding and border within the specified width/height, making layout calculations more predictable.
