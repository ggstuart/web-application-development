> Setting the `display` property of an element to `inline` or `block` will change the way the element participates in the **flow** layout.
> The default value is `inline`.

{{<iframe src="examples/outer-display" width="1000" height="200">}}{{</iframe>}}

> A `<span>` element has no default styles.
> The user-agent stylesheet sets some elements (e.g. `<div>`, `<p>`) to `display: block` by default.
